# Rock Paper Scissor Project

## Overview
This project will have me utilize the tools I have learned to
automate the classic game of Rock Paper Scissors (sometimes referred to as Roshambo). This will be done without using AI or looking up previous solutions.

## Features
Uses based HTML structure
Primarily uses the console to request the object and display results

* JavaScript

* Git/Github
* CLI
